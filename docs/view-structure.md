# Z-CMS 뷰/탭 구조 요약

## 1. 통합 관제 대시보드 (`home`)
KPI 요약 카드 + 트렌드 차트. 탭 없음 (단일 페이지).

## 2. 수익 분석 (`profit`)
| 탭 | 목적 |
|---|---|
| 채널별 수익 | 판매 채널별 매출/이익 비교 |
| 품목별 랭킹 | 수익 기여도 순위 |
| 수익 트렌드 | 기간별 수익 추이 차트 |
| 예산 대비 실적 | 예산 vs 실적 달성률 |
| 현금 흐름 | 현금 유입/유출 현황 |

## 3. 매출 분석 (`sales`)
| 탭 | 목적 |
|---|---|
| 매출 트렌드 | 일/주/월별 매출 추이 |
| 베스트셀러 | 판매량 상위 품목 |
| 급상승/급하락 | 매출 변동 이상 품목 감지 |
| 수익성 매트릭스 | 매출액-마진율 2축 분포 |
| 채널 심층분석 | 채널별 매출 상세 비교 |

## 4. 원가 관리 (`cost`)
| 탭 | 목적 |
|---|---|
| 원가 총괄 | 원가 구성 요약 (재료+노무+경비) |
| 원재료 | 주요 원재료 단가/사용량 분석 |
| 부재료 | 부재료 비용 추적 |
| 노무비 | 인건비 현황 및 추이 |
| 수도광열전력 | 유틸리티 비용 관리 |

## 5. 생산/BOM 관리 (`production`)
| 탭 | 목적 |
|---|---|
| 생산 현황 | 일별 생산 실적 모니터링 |
| 폐기 분석 | 폐기량/폐기율 추적 |
| 생산성 분석 | 라인별 생산 효율 비교 |
| BOM 이상 감지 | 표준 BOM 대비 이상 항목 |
| BOM 오차 | 표준 vs 실제 투입량 차이 |
| 수율 추적 | 품목별 수율 모니터링 |

## 6. 재고/발주 관리 (`inventory`)
| 탭 | 목적 |
|---|---|
| 재고 현황 | 안전재고 대비 현재고 |
| 이상징후 분석 | 재고 이상 패턴 AI 감지 |
| 통계적 발주 | 수요예측 기반 최적 발주량 |
| 발주 분석 | 발주 이력 및 납기 분석 |
| 재고비용 | 재고 유지비용 산출 |

## 7. 설정 (`settings`)
카테고리별 접이식 섹션 (탭 아님):

| 카테고리 | 목적 |
|---|---|
| 데이터 연결 | ECOUNT API, 구글시트 연동 |
| 원가/비용 관리 | 원가 기준, 노무비, 예산 설정 |
| 채널/매출 관리 | 채널 수수료, 정산주기 설정 |
| 재고/발주 관리 | ABC-XYZ 분류, 발주 파라미터 |
| 분석/감지 설정 | AI, 이상 감지 임계값 |
| 경영 목표/시스템 | 독립채산제 목표, 설정 백업 |
